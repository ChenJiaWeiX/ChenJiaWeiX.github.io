# MarkDown（MD）

## MarkDown 的介绍

- 它是对内容进行排版的语法工具。用一两个符号，就能实现很好的排版效果。
- 只需要把 .txt 文档的后缀改成 .md 格式，就能在网页或者辅助工具上看到 MarkDown 的排版效果。
